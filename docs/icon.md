## 插槽

| 名称     | 介绍            |
| -------- | --------------- |
| 默认插槽 | 插入svg图标标签 |

## 属性

| 名称 | 默认值 | 是否同步 | 介绍         | 类型                  |
| ---- | ------ | -------- | ------------ | --------------------- |
| name | 无     | 是       | 内置图标名称 | string (查看原型定义) |

> [!NOTE]
>
> 在同时有`name`属性和插槽元素时，将默认使用插槽元素作为`svg`图标。使用了不存在的`name`时会在控制台打印错误。
